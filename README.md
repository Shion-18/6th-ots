# オープンチームシート - 第6世代ポケモン対戦用

第6世代（XY/ORAS）ポケモン対戦のオープンチームシートルール用Webアプリケーション

## 概要

オープンチームシートルールで対戦する際に、お互いのパーティ情報をスムーズに共有・確認できるWebアプリです。

### 主な機能

- **詳細なポケモン編集**
  - 全ポケモンから選択可能
  - 努力値・個体値の詳細設定
  - 性格・特性・持ち物の選択
  - 技の自由な組み合わせ
  - リアルタイム実数値計算

- **パーティ管理**
  - LocalStorageによる複数パーティの保存
  - 保存済みパーティの読み込み・削除
  - パーティ名のカスタマイズ

- **共有機能**
  - URL共有（Base64エンコード）
  - 2人のユーザー間でのパーティ確認
  - 自分のパーティは自分だけが編集可能

- **レスポンシブデザイン**
  - スマートフォン完全対応
  - 第9世代風のモダンUI

## 技術スタック

- **フロントエンド**: Next.js 16.1.1 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **デプロイ**: Vercel

## セットアップ

### 前提条件

- Node.js 18以上
- npm または yarn

### インストール

```bash
# リポジトリをクローン
git clone <repository-url>
cd 6thots-app

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてアプリケーションにアクセスできます。

### ビルド

```bash
# プロダクションビルド
npm run build

# プロダクションサーバーを起動
npm start
```

## 使い方

### 1. パーティを作成

1. トップページから「パーティを作成」をクリック
2. 「ポケモンを追加」ボタンをクリック
3. ポケモン編集画面で以下を設定:
   - ポケモンの選択（検索可能）
   - レベル（50 or 100）
   - 特性・持ち物・性格
   - 努力値（合計510まで）
   - 個体値（0-31）
   - 技（最大4つ）
4. 実数値が自動計算されて表示される
5. 「保存」で確定

### 2. パーティを保存（自分専用）

1. パーティ名を入力
2. 「保存」ボタンでLocalStorageに保存
3. 「マイパーティ」から保存済みパーティを確認・管理

### 3. パーティを共有（対戦相手用）

1. 「共有URL生成」ボタンをクリック
2. 生成されたURLをコピー
3. 対戦相手にURLを送信

### 4. 相手のパーティを確認

1. トップページの「相手のパーティ」をクリック
2. 相手から受け取ったURLを貼り付け
3. 相手のパーティ情報が表示される

### 5. 対戦開始

- お互いのパーティ情報を確認して、オープンチームシートルールで対戦開始

│   ├── gen6-pokemon.json  # 第6世代ポケモンデータ（15体）

    └── pokemon.ts        # Pokemon、Team等の型定義

## 2人のユーザー間でのパーティ共有の仕組み

### 自分のパーティ（編集可能）
- LocalStorageに保存
- ブラウザごとに独立して管理
- 「マイパーティ」ページで確認・編集・削除が可能
- 他のユーザーには見えない

### 相手のパーティ（閲覧のみ）
- URL共有機能を使用
- Base64エンコードされたパーティデータがURLに含まれる
- URLを開くだけで相手のパーティを確認可能
- 編集不可（閲覧専用）

### オープンチームシートの流れ
1. プレイヤーA: 自分のパーティを作成→「保存」（LocalStorage）
2. プレイヤーA: 「共有URL生成」→URLをコピー→プレイヤーBに送信
3. プレイヤーB: 同様に自分のパーティを作成→URLをプレイヤーAに送信
4. プレイヤーA＆B: お互いのURLを開いて相手のパーティを確認
5. 対戦開始
