# オープンチームシート - 第6世代ポケモン対戦用

第6世代（XY/ORAS）ポケモン対戦のオープンチームシートルール用Webアプリケーション

## 概要

オープンチームシートルールで対戦する際に、お互いのパーティ情報をスムーズに共有・確認できるWebアプリです。

### 主な機能

- 詳細なポケモン編集
  - 第６世代までの全ポケモンから選択可能
  - 性格・特性・持ち物の選択

- パーティ管理
  - 保存済みパーティの読み込み・削除
  - パーティ名のカスタマイズ

- 共有機能
  - URL共有（Base64エンコード）
  - 2人のユーザー間でのパーティ確認
  - 自分のパーティは自分だけが編集可能

- レスポンシブデザイン
  - スマートフォン完全対応
  - 第9世代風のモダンUI


## 技術スタック

- フロントエンド: Next.js 16.1.1 (App Router)
- 言語: TypeScript
- スタイリング: Tailwind CSS
- デプロイ: Vercel


## 使い方

### 1. パーティを作成

1. トップページから「パーティを作成」をクリック
2. 「ポケモンを追加」ボタンをクリック
3. ポケモン編集画面で以下を設定:
   - ポケモンの選択（検索可能）
   - レベル（50 or 100）
   - 特性・持ち物・性格
   - 努力値（合計510まで）
   - 個体値（0-31）
   - 技（最大4つ）
4. 実数値が自動計算されて表示される
5. 「保存」で確定

### 2. パーティを保存（自分専用）

1. パーティ名を入力
2. 「保存」ボタンでLocalStorageに保存
3. 「マイパーティ」から保存済みパーティを確認・管理

### 3. パーティを共有（対戦相手用）

1. 「共有URL生成」ボタンをクリック
2. 生成されたURLをコピー
3. 対戦相手にURLを送信

### 4. 相手のパーティを確認

1. トップページの「相手のパーティ」をクリック
2. 相手から受け取ったURLを貼り付け
3. 相手のパーティ情報が表示される

### 5. 対戦開始

- お互いのパーティ情報を確認して、オープンチームシートルールで対戦開始


## プロジェクト構造

```
6thots-app/
├── app/                    # Next.js App Router
│   ├── page.tsx           # ホームページ
│   ├── builder/           # パーティビルダー（作成・編集）
│   ├── my-teams/          # マイパーティ管理
│   └── view/              # パーティ閲覧（URL共有）
├── components/            # Reactコンポーネント
│   └── ui/
│       ├── PokemonCard.tsx    # ポケモンカード表示
│       ├── PokemonEditor.tsx  # ポケモン詳細編集モーダル
│       └── TeamView.tsx       # パーティ表示
├── data/                  # 静的データ
│   ├── gen6-pokemon.json  # 第6世代ポケモンデータ（15体）
│   ├── moves.json         # 技データ（22種）
│   └── items.json         # 持ち物データ（18種）
├── lib/                   # ユーティリティ関数
│   ├── stats.ts          # ステータス計算・努力値検証
│   ├── team-encoder.ts   # URL共有・LocalStorage管理
│   └── type-colors.ts    # タイプカラー定義
└── types/                 # TypeScript型定義
    └── pokemon.ts        # Pokemon、Team等の型定義
```
